Stencil AR pour Meta Quest 3

Projetez des pochoirs d’images sur des surfaces réelles en réalité mixte.

1) Charger une image (JPEG/PNG)

Glissez-déposez une image ici ou utilisez le bouton de sélection. PNG avec transparence supporté.

Déposez votre image ici

Astuce: L’image est stockée localement (aucun envoi). Taille optimisée automatiquement ≤ 2048 px.

2) Lancer l’application AR

Sur votre Quest 3, ouvrez cette page via GitHub Pages (HTTPS). Puis démarrez l’AR:

  1. Chargez une image ci-dessus (optionnel, une image par défaut est fournie).
  2. Cliquez sur « Ouvrir l’application ».
  3. Dans la page suivante, appuyez sur « Entrer en VR » pour démarrer l’AR.
Ouvrir l’application

Contrôles (Quest)

  • Trigger: Sélection / placement
  • Grip: Rotation du stencil
  • Thumbstick: Échelle (Y) et distance (X)
  • Boutons X/Y: Basculer Auto / Manuel
  • Boutons A/B: Opacité - / +

Projet statique — fonctionne en HTTPS. Voir README pour GitHub Pages et serveur local.
